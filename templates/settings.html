Discord Notifications

Enter your Discord webhook URL to receive notifications when containers are restarted, started, or stopped.

How to get a Discord Webhook URL:

  1. Open your Discord server settings
  2. Go to Integrations - Webhooks
  3. Click New Webhook
  4. Give it a name (e.g., "Chrontainer")
  5. Select the channel where you want notifications
  6. Click Copy Webhook URL
  7. Paste the URL in the field above

Usage Examples

curl

curl -H "X-API-Key: chron_xxxxx" http://your-server:5000/api/containers

Python

import requests
response = requests.get(
    'http://your-server:5000/api/containers',
    headers={'X-API-Key': 'chron_xxxxx'}
)

Home Assistant REST Command

rest_command:
  restart_container:
    url: "http://your-server:5000/api/container/{{ container_id }}/restart"
    method: POST
    headers:
      X-API-Key: "chron_xxxxx"

Usage Examples

curl (GET or POST)

curl "http://your-server:5000/webhook/YOUR_TOKEN"

curl with container override

curl "http://your-server:5000/webhook/YOUR_TOKEN?container_id=abc123&host_id=1"

Home Assistant Automation

automation:
  - alias: "Restart media server at 3am"
    trigger:
      platform: time
      at: "03:00:00"
    action:
      service: rest_command.trigger_webhook
      data:
        url: "http://chrontainer:5000/webhook/YOUR_TOKEN"
